# GP Autocare — Landing Page

A clean, single-page landing site for **GP Autocare**, a professional auto detailing shop that brings back that new-car feeling.

## Features

- **Hero** — GP monogram badge, brand name, and tagline
- **About** — what the shop does and who it's for, with a "Get in touch" call to action
- **Footer** — contact info and social links
- **Dark / light mode** — toggle in the top-right corner; respects OS preference and remembers your choice

## Brand colors

| Role | Name | Hex |
| --- | --- | --- |
| Primary | Ink | `#10151C` |
| Accent | Azure | `#2E8BE0` |
| Neutral | Cream | `#F3EFE7` |

## Running locally

It's a single static file — open `index.html` directly, or serve it:

```bash
python3 -m http.server 8000
```

Then visit http://localhost:8000/.
